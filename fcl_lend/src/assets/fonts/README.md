# Шрифты для проекта FCL

Для корректной работы проекта необходимо добавить следующие файлы шрифтов в эту папку:

## Необходимые шрифты:

1. **Cygre Regular** (`Cygre-Regular.woff2`, `Cygre-Regular.woff`)
2. **Cygre ExtraBold** (`Cygre-ExtraBold.woff2`, `Cygre-ExtraBold.woff`)
3. **Terminator Genisys** (`TerminatorGenisys.woff2`, `TerminatorGenisys.woff`)
4. **Azoft Sans** (`AzoftSans-Regular.woff2`, `AzoftSans-Regular.woff`)

## Как получить шрифты:

1. Откройте макет в Figma
2. Выберите любой текстовый элемент
3. В правой панели Properties найдите раздел "Text" → "Font"
4. Нажмите на название шрифта и выберите "Download font" или экспортируйте через меню

Альтернативно, можно использовать похожие системные шрифты как fallback:
- Cygre → можно заменить на Inter или Roboto
- Terminator Genisys → можно заменить на Impact или Arial Black
- Azoft Sans → можно заменить на Arial или Helvetica
